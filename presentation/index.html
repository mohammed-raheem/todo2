Todo List Application

A modern, responsive web app built with React, TypeScript, and Redux Toolkit. It demonstrates best practices in React development, state management, and testing while providing users with an efficient way to manage their tasks.

Features

  • User authentication (login/logout)
  • Add, edit, and delete tasks
  • Mark tasks as complete/incomplete
  • Filter tasks by status (All, Completed, Pending)
  • Responsive design for mobile and desktop
  • Persistent storage using localStorage
  • Dark mode toggle
  • Multilingual support (English and Arabic)

Technologies and Techniques Used

  • React 18 with TypeScript
  • Redux Toolkit for state management
  • React Router for navigation
  • Material-UI (MUI) for UI components
  • i18next for internationalization
  • Vite as the build tool
  • Responsive design techniques
  • Local storage for data persistence

Upcoming Features

  • User registration functionality
  • Task categories or tags
  • Due dates and reminders for tasks
  • Drag and drop functionality for task reordering
